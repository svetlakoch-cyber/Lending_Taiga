Фирменные комплексы линии «Тайга»

В основе средств линии используются природные компоненты хвойных растений Сибири и фирменные комплексы бренда.

Бальзам «Горная тайга»

100% натуральный продукт, не содержит искусственных добавок. В состав входят: экстракт пихты сибирской, экстракт сосны кедровой сибирской. Натуральные компоненты выделены из экологически чистого сырья, собираемого специально для приготовления бальзама.

Комплекс «Эфирка»

Сбалансированный природный комплекс, содержащий только натуральные биологически активные вещества. Произведен технологией сверхкритической флюидной экстракции, без температурной обработки, что делает этот продукт существенно богаче по составу, чем обычные эфирные хвойные масла.
